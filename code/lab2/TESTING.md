LAB2 - 调度器测试说明
=====================

已添加的调度器测试功能
----------------------

1. 优先级测试 (PriorityTest)
   - 命令: ./n2 -P
   - 功能: 创建不同优先级的线程（10、50、25），验证高优先级线程优先执行

2. 复杂优先级测试 (ComplexPriorityTest) 
   - 命令: ./n2 -C
   - 功能: 测试线程在执行过程中动态改变优先级的情况

3. 饥饿测试 (StarvationTest)
   - 命令: ./n2 -S
   - 功能: 测试低优先级线程是否会被高优先级线程饿死

文件变更
--------
- schedtest.cc: 实现了上述三个测试函数
- Makefile.local: 添加了schedtest.cc到编译列表
- main.cc: 添加了命令行参数处理，支持-P、-C、-S选项

注意：由于编译环境问题，需要使用预编译的n2可执行文件运行测试。
